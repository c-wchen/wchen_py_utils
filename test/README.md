##inspect 模块属性
http://www.codebaoku.com/it-python/it-python-231742.html

通过inspect.getmembers(object[, predicate])返回模块属性
其中包括
1. 模块的内省
2. 对模块中的类进行内省
3. 对类中的方法/功能的内省
4. 内省一类的对象
5. 检索类的来源
6. 检索方法/功能的来源
7. 获取方法签名
8. 类字符串的文档
9. 内省类层次结构
10. 在运行时环境中对框架和堆栈进行内省
